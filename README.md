# Jobby App

A job portal web application built with **React.js**, featuring authentication, protected routes, and dynamic job listings.

## Purpose
This project demonstrates my ability to apply real-world web development concepts:
- Implementing **JWT authentication** and secure login flows
- Managing **protected routes** with React Router
- Integrating **REST APIs** for profile and job data
- Handling **error states and failure views**
- Building reusable components with clean state management

## Features
- Secure login with JWT and cookies
- Profile and job data fetched from APIs
- Job search with filters (employment type, salary range, keyword)
- Detailed job view with company logo, skills, and "life at company"
- Responsive design and accessibility compliance

## Tech Stack
- React.js
- React Router
- CSS
- REST APIs
- JWT Authentication
